# Space Portfolio Texture Prompt Library

Use these prompts with the image model for production texture candidates. Export sources large, then crop and compress to WebP or AVIF before putting them in `public/textures/`.

## Black Hole Hero

Create a cinematic astrophotography-style black hole accretion disk texture plate, deep black center, realistic hot orange and pale gold gas, subtle blue rim light, high dynamic range, no text, no UI, no borders, no logo, no obvious symmetry, wide negative space around the object, suitable for compositing in Three.js.

## Nebula Wide Background

Create a wide deep-space nebula texture plate, indigo black background, controlled blue violet and soft magenta gas clouds, sparse bright stars, realistic dust structure, high dynamic range, no text, no UI, no planet, no spacecraft, no framing artifacts, designed as a subtle layered background.

## Accretion Flow Texture

Create an abstract horizontal accretion-flow texture, luminous orange gold plasma streaks, wispy gas turbulence, transparent-looking dark gaps, realistic astrophysical light behavior, no hard circular object, no text, no logo, no UI, seamless-feeling edges for shader use.

## Dust And Gas Alpha Plate

Create a monochrome alpha-mask style cosmic dust plate, white smoky particles and faint gas wisps on pure black, high detail, sparse center, no stars larger than pinpoints, no text, no UI, designed for additive blending in a WebGL particle scene.

## Abstract Cosmic Signal Glyphs

Create a set of crisp abstract cosmic signal glyph marks, thin white geometric linework on black, inspired by observatory interfaces and star-map notation, no letters, no numbers, no recognizable logos, sharp edges, high contrast, suitable for vector tracing.

## OG Social Preview

Create a cinematic portfolio social preview image for Warre Snaet, black hole and starfield composition, deep black with orange accretion glow and blue violet nebula, clean negative space on left for real text to be added later, no generated text, no UI, no logo.
